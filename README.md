# Customer Behavior & Sales Insights Dashboard

## Overview

This project presents an end-to-end data analysis workflow to understand customer behavior, purchasing patterns, and sales trends. The analysis was performed using Python, MySQL-XAMPP, and Power BI.

## Tools & Technologies

* Python (Jupyter Notebook) – Data preprocessing and cleaning
* XAMPP-MySQL – Data storage and querying
* Power BI – Data visualization and dashboard creation

## Workflow

1. Data Preprocessing

   * Cleaned dataset by handling missing values and duplicates
   * Performed data transformation using Python

2. Data Analysis

   * Imported data into MySQL
   * Executed SQL queries to analyze customer behavior

3. Data Visualization

   * Built an interactive Power BI dashboard
   * Included KPIs, category analysis, seasonal trends, and customer segmentation
   * 

## Key Insights

* Clothing category generates the highest revenue
* Young adult customers contribute significantly to overall sales
* Subscribed customers tend to purchase more frequently
* Seasonal trends impact average purchase behavior
